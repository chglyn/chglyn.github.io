简介

TypeScript 是 JavaScript 的超集，是一种编程语言，包含静态类型和面向对象。

适用于多人协作和大型应用。

TypeScript vs JavaScript

  • 应用程序。

  • 是 JS 超集。

  • 跨平台，且开源。

  • 开始于 JS，终止于 JS。

  • 重用 JS，甚至可以引入 JS 流行的库。

  • TS 提供类，接口， 模块。
